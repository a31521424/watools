## LLM 必读

如果任务和插件开发相关,阅读顺序应为:

1. [`PLUGIN_DEVELOPMENT_INDEX.md`](./PLUGIN_DEVELOPMENT_INDEX.md)
2. 只打开索引里明确相关的章节模块

不要一开始就完整阅读所有插件文档,除非任务明确要求全面审阅规范。
